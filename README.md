
### V2.0 版本

算是完整的一个版本 ，完全可以使用，而且进行了打包测试，之前没有打包测试 ，有一些问题 没发现
使用 QFramework HybirdCLR YooAsset 再加上 Python Server 进行了 Unity模拟测试 、打包测试 和 Host 远程测试，算是一个模板，有一些植物大战僵尸的部分，但是不多，而且植物大战僵尸的源码全部都是热更代码，可以删除

Unity 2022.3.62f1
QFramework v1.0.187 Extensions v0.0.9
YooAsset 2.3.18
HybridCLR 8.9.0
Python 3.11.0

打包之后的exe文件夹 附件也放在这里 可以测试

目前实现的逻辑：
1. 进入之后会先 加载热更新资源，加载完成之后进入主页面场景
2. 主场景目前只有一个按钮，按钮点开是关卡选择界面，因为关卡逻辑没做完，所以只能玩第一关，总共会做5关
3. 选择1-1 之后会进入第一个关卡场景（所有场景都一样，关卡只是难度不同）
4. 可以种植豌豆射手，也会有太阳生成，也可以铲子铲掉植物
5. 僵尸会生成往左边走，目前bug 会导致 僵尸走到中间触发吃了脑子，后续会修复
6. 关卡结束没做，因为和关卡里面的新植物解锁和新关卡解锁 相关联就没做
